# Instrucciones básicas para clonar e instalar
Si solo queremos los archivos descargamos el proyecto comprimido desde el enlace dandole a **<> Code** y luego a **Download ZIP** y lo descomprimimos en **C:\xampp\htdocs**

*o*

Si queremos clonar el repositorio entramos a **NetBeans** elegimos la opción **Team**, luego nos vamos a **Git** y por último a **Clone...**, allí introducimos el enlace del proyecto de GitHub sin usuario ni contraseña y elegimos el repositorio donde clonarlo, elegimos la rama y confirmamos la localización del archivo, una vez terminado el proyecto ya estará disponible
